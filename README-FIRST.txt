PRECISION HYDRAULIC — PWA TEST EDITION

IMPORTANT
This package must be hosted over HTTPS. Do not open index.html directly from Mail or the Files app.

QUICKEST TEST WITH NETLIFY DROP
1. On a computer, unzip this package.
2. Open https://app.netlify.com/drop in a browser.
3. Drag the entire Precision_Hydraulic_PWA_Test_Edition folder onto the page.
4. Netlify will provide an HTTPS address.
5. Open that address in Safari on the iPhone.
6. Tap Share > Add to Home Screen > Add.
7. Launch Precision Hydraulic from the new Home Screen icon.

DATA WARNING
This test edition still stores job data separately in each browser/device. Jobs entered on the computer will not automatically appear on the iPhone. Export/import JSON remains the transfer method until a shared database is added.

UPDATES
After uploading a newer version, Safari may briefly use the cached copy. Close and reopen the app. If needed, remove the Home Screen app, clear Safari website data for the test address, and add it again.
